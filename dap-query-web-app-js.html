Canvas DAP Query Web App

About This Application

This web application allows you to query and download data from the Canvas Data Access Platform (DAP) API. The application runs entirely in your browser using JavaScript, which means:

  • Your credentials are never sent to a third-party server
  • Downloaded data is saved directly to your computer
  • No server-side processing or storage is required
Note: This application uses the File System Access API to save files, which is only supported in Chromium-based browsers (Chrome, Edge, Opera, etc.).

Browser Compatibility

This application uses the File System Access API which is currently supported in:

  • Chrome 86+
  • Edge 86+
  • Opera 72+

It is not supported in Firefox or Safari.

This application was built with JavaScript and runs entirely in your browser.

No data is sent to any server other than Canvas DAP API.
